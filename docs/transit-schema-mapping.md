# Відповідність таблиць бази Test та TypeORM сутностей

Нижче наведено відповідність між початковими таблицями в базі даних `Test` (українські назви/поля) та поточними назвами таблиць і колонок у TypeORM сутностях.

## Мапінг таблиць

| Postgres таблиця (укр)        | Поточна таблиця (англ) |
| ----------------------------- | ---------------------- |
| користувачі                   | users                  |
| водії                         | drivers                |
| зупинки                       | stops                  |
| типи_транспорту               | transport_types        |
| маршрути                      | routes                 |
| зупинки_маршрутів             | route_stops            |
| точки_маршрутів               | route_points           |
| транспорти                    | vehicles               |
| призначення*водіїв*транспорту | driver_assignments     |
| розклади                      | schedules              |
| рейси                         | trips                  |
| транспортні_картки            | transport_cards        |
| поповнення_карток             | card_top_ups           |
| квитки                        | tickets                |
| штрафи                        | fines                  |
| оскарження_штрафів            | fine_appeals           |
| скарги_пропозиції             | complaints             |
| логи*gps*користувачів         | user_gps_logs          |
| логи*gps*транспорту           | vehicle_gps_logs       |

## Поля: приклад

Нижче приклад для двох таблиць. За потреби можеш доповнити список іншими.

### 1. Таблиця користувачі → users

| Українська назва поля | TypeORM колонка | Тип даних                     |
| --------------------- | --------------- | ----------------------------- |
| id                    | id              | `integer`                     |
| email                 | email           | `text`                        |
| телефон               | phone           | `text`                        |
| піб                   | full_name       | `text`                        |
| дата_реєстрації       | registered_at   | `timestamp without time zone` |

### 2. Таблиця маршрути → routes

| Українська назва поля | TypeORM колонка   | Тип даних     |
| --------------------- | ----------------- | ------------- |
| id                    | id                | `bigint`      |
| тип_транспорту_id     | transport_type_id | `bigint` (FK) |
| номер                 | number            | `text`        |
| напрямок              | direction         | `text`        |
| активний              | is_active         | `boolean`     |

> Якщо потрібно зафіксувати всі поля, продовж цей документ за аналогією для решти таблиць.
